# PipeRescue
Pipe Rescue playable ad assessment project

1.What it is: A mobile playable ad built in Cocos Creator 3.8.6 where the player rotates pipe tiles to connect a water source to a goal.

2.How to run: Open the project in Cocos Creator 3.8.6,[main scripts are PipeGame.ts & TileData.ts] it opens in web-browser using a localhost.(LocalhostID : http://localhost:7456/) 

3.Cocos Creator version: 3.8.6

4.How to play: Tap blue pipe tiles to rotate them 90 degrees. Connect the green S tile to the gold E tile.

5.Move limit: Player has 12 moves to solve the puzzle.

6.Win condition: Connect S to E — "You fixed it!" screen appears.

7.Fail condition: Run out of moves — "Out of Moves!" screen appears with a Retry option.

8.Buttons: Check Path verifies connection. Play Now and Retry both restart the game.

9.AI tools used: AI tools were used for code assistance, debugging, and development guidance throughout the project.

10. Known issues: Canvas design resolution appears landscape in the editor but renders correctly in the browser at portrait size.
